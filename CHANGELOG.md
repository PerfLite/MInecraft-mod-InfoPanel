## Версия 1.2 — Лог предметов, звуки портала, новый GUI

### Лог поднятых предметов (SkyUI-стиль)

- Показывает поднятые предметы в углу экрана: `+3 🪨`
- Формат: зелёный счётчик `+N` + иконка предмета
- Одинаковые предметы **объединяются** в одну строку
- **Анимация исчезновения** — плавное угасание в конце жизни
- Кастомный фон: **острый левый угол**, **закруглённый правый**
- Позиция настраивается через **F8** (фиолетовая рамка в редакторе)
- Настройки (вкладка Строки):
  - Вкл/выкл
  - Прозрачность фона (0–255)
  - Максимум записей (1–20)
  - Время показа (2–20 секунд)
  - Масштаб (0.5×–2.0×)
  - Сброс позиции

### Звуки портала

Новая вкладка **"Звуки"** в настройках:

| Настройка | Звук |
|-----------|------|
| Гул портала (вкл/выкл + громкость 0–100%) | `block.portal.ambient` |
| Переход через портал (вкл/выкл + громкость 0–100%) | `block.portal.trigger` + `block.portal.travel` |

---

### Новый GUI настроек

Полностью переработан:

- **Левая панель навигации** с иконками категорий: `⚙ ☰ ◎ ⬡ ♪`
- Тёмный фон без стандартной MC-текстуры
- Кнопки **по ширине текста** (не растянуты)
- Кнопки позиции с индикатором `●`/`○`
- **Y** — открыть/закрыть (повторное нажатие закрывает)
- **Esc** — закрыть
- Три отдельные кнопки сброса позиций: панель / компас / прочность

## Версия 1.1 — WAILA, HUD, локализация

### Локализация

- Весь UI переведён на **русский и английский**
- Все строки в `assets/infopanel/lang/en_us.json` и `ru_ru.json`

---

### HUD-панель

- **Направление** — оси координат вместо букв: `Север (+Z)`, `Восток (+X)` и т.д.
- **Игровое время** — иконки солнца/луны (`icon_sun.png`, `icon_moon.png`)

---

### WAILA

#### Блоки
- **Растения** — уровень созревания: `Рост: 4 : 7 ✔ Созрело`
- **Редстоун-провод** — `Сигнал: 11 : 15`
- **Повторитель** — `Задержка: 2 ticks [ВКЛ]`
- **Компаратор** — режим + состояние + сигнал
- **Печи / Плавильни / Коптильни** *(синглплейер)* — иконки предметов + прогресс-бар + количество топлива
- **Ульи / Гнёзда** — `Мёд: 3/5` + `Пчёл: 2/3`
- **Котлы** — тип содержимого + уровень
- **Координаты портала** — конвертация Overworld ↔ Nether × 8

#### Мобы
- **Портрет** — 3D-рендер через `InventoryScreen`
- **Жители** — профессия с цветом по уровню (серый→белый→зелёный→голубой→золотой), уровень в скобках

---

### Редактор позиций (F8)

- Убрано размытие экрана
- Переведены подсказки и метки

---

### Технические изменения

- `Lang.java` вынесен в отдельный файл
- `PortalSoundHandler.java` — перехват звуков через `PlaySoundEvent`
- Данные печей и ульев читаются через server thread
- Уровень жителя читается напрямую с серверной сущности
- `sectionHeaders.clear()` в `init()` — исправлено накопление заголовков
